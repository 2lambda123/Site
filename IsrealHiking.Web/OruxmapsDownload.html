Israel Hiking and Biking Maps and OruxMaps

Install or update using an Android device connected to the web

Click on one of the links below from an Android device with OruxMaps 6.5.4 or later. The download may take several minutes, and afterwards you can enjoy the map with no need for a network connection.

  • {{link.description}}

Note:

  • After the installation please refresh the offline maps by:
    Maps → Switch map → OFFLINE → Refresh (the counter-clockwise arrows)
  • The free space required for direct installation on an Android device is twice the size of the downloaded map.

Install or update using a PC connected to your Android device with a USB cable

  • Click one of the above link for the requested map and wait for the zip file to finish downloading.
  • Connect the Android device to your PC.
  • If needed, change your Android device USB connection mode from "USB for charging" to "File transfers".
  • On your PC:
    • Open the downloaded zip file and copy the directory it contains to a temporary locaton, such as your Desktop.
    • Move this directory into the "Oruxmaps\mapfiles" directory of the connected Android device.
  • Open OruxMaps and refresh the offline maps by:
    Maps → Switch map → OFFLINE → Refresh (the counter-clockwise arrows)

מפות הטיולים והאופניים הפתוחות ו- OruxMaps

התקנה או עדכון בעזרת מכשיר אנדרואיד המחובר לרשת

לחצו על אחד הקישורים הבאים ממכשיר אנדרואיד שבו מותקנת OruxMaps מגרסה 6.5.4 או יותר. ההורדה עשויה להמשך דקות רבות, אך בסיומה תוכלו להשתמש במפה ללא צורך בגישה לרשת.

  • {{link.descriptionHebrew}}

שימו לב:

  • בסיום ההתקנה יש לעדכן את רשימת המפות הלא-מקוונות על ידי:
    מפות ← החלף מפה ← Offline ← רענן (חיצים מסתובבים)
  • המקום הפנוי הנדרש להתקנה ישירה במכשיר אנדרואיד הוא כפול מגודל המפה.

התקנה או עדכון באמצעות מחשב אישי שמחובר למכשיר האנדרואיד עם כבל USB

  • לחצו על אחד הקישורים למעלה עבור המפה הדרושה והמתינו לסיום הורדת קובץ ה zip.
  • חברו את מכשיר האנדרואיד עם כבל USB.
  • במידת הצורך, במכשיר האנדרואיד שנו את מצב חיבור ה USB מ "טעינה" ל "העברת קבצים".
  • במחשב האישי:
    • פתחו את קובץ ה zip והעתיקו את התיקיה שהוא מכיל למקום זמני כגון שולחן העבודה שלכם.
    • העבירו את התיקיה שהעתקתם לתיקית "Oruxmaps\mapfiles" שבמכשיר האנדרואיד.
  • פתחו את OruxMaps ורעננו את רשימת המפות הלא-מקוונות על ידי:
    מפות ← החלף מפה ← Offline ← רענן (חיצים מסתובבים)
