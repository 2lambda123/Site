This map was generated from Open Street Map (OSM) data where it can be viewed and edited. All you need to do is create an account and start mapping. Note that the changes will not affect this map instantly.

Below are some links related to this project

  • See our map on Github for instructions on how to create this map by yourself and links to map downloads.
  • See our site on Github in order to submit a change request an idea or an issue you encountered, this will help us improve your experience.
  • See our Facebook group in order to chat with users just like you.
  • See Israel OSM Wiki Project for common rules of the Israeli Open Street Map community.
  • See Israel OSM Forum for the Israeli Open Street Map forum where you can post questions and look for answers.
  • See Oruxmaps folder listing in order to get an offline map. more instructions can be found here
  • See Oruxmaps android direct download in order to get an offline map straight to your android device.
